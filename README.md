# Requirements (https://wdavidcalsin.vercel.app/)

Node.js v14.12.0 LTS & NPM v6.x



### Quick start

To prepare and start your dev enviroment

1. Install dependencies `npm install`
2. Run server `npm run dev`
3. Go to `http://127.0.0.1:3000`
